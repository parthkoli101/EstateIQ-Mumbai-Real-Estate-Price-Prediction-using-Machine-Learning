Architecture

Training
pipeline.

The model follows a classic supervised regression pipeline — load, clean, scale, split, train, serialize. The same scaler used during training is applied at inference time, ensuring consistent feature distributions.

Step 01
Load & Clean Data
Mumbai housing CSV loaded. Locality names normalized. Prices, price-per-sqft, and area filtered to realistic ranges (₹10L–₹10Cr / ₹2K–₹40K sqft / 250–5000 sq ft).
Step 02
Feature Engineering
Locality median price-per-sqft computed and merged as a feature. Missing values imputed with column medians. 7 final features selected.
Step 03
StandardScaler Preprocessing
All 7 features scaled using StandardScaler (zero mean, unit variance). Scaler fit only on training data. Saved as mumbai_scaler.pkl.
Step 04
Train RandomForestRegressor
200 trees, max_depth=15, random_state=42. Trained on 80% split. Achieves R²=0.9995 on holdout test set. Saved as mumbai_price_model.pkl.
Step 05
Flask Inference API
Model + scaler loaded at Flask startup. POST /predict scales input → predicts → returns formatted Crore/Lakh result via JSON.

# Train the Random Forest model
from sklearn.ensemble import RandomForestRegressor
from sklearn.preprocessing import StandardScaler
import pickle

# Scale features
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Train/test split
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y,
    test_size=0.2,
    random_state=42
)

# Random Forest with 200 trees
model = RandomForestRegressor(
    n_estimators=200,
    max_depth=15,
    random_state=42
)
model.fit(X_train, y_train)

# Save model and scaler
pickle.dump(model, open("mumbai_price_model.pkl", "wb"))
pickle.dump(scaler, open("mumbai_scaler.pkl", "wb"))
app.py — /predict endpoint
# Flask prediction route
@app.route('/predict', methods=['POST'])
def predict():
    data = request.get_json()

    # Build feature vector (7 features)
    features = np.array([[
        data['area'],
        data['bedroom_num'],
        data['bathroom_num'],
        data['balcony_num'],
        data['age'],
        data['total_floors'],
        price_per_sqft  # from locality map
    ]])

    # Scale → Predict
    scaled = scaler.transform(features)
    price  = model.predict(scaled)[0]

    return jsonify({
        'predicted_price': price,
        'formatted_price': format_indian(price)
    })
