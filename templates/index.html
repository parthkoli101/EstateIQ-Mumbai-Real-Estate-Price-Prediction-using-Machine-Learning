Under the Hood

How EstateIQ thinks.

A complete ML pipeline — from raw Mumbai housing data to a Rupee prediction — in five transparent steps.

01

Data Ingestion & Cleaning

The Mumbai housing CSV is loaded and cleaned. Locality names are standardised. Outlier prices below ₹10L or above ₹10Cr are removed, as are extreme area and price-per-sqft values. Missing numerics are median-imputed.

# Load & clean Mumbai data df = pd.read_csv("mumbai_house_price.csv") df['locality'] = df['locality'].str.title() # Remove price outliers df = df[(df['price'] > 1_000_000) & (df['price'] < 100_000_000)] df = df[(df['area'] > 250) & (df['area'] < 5000)]
02

Locality Encoding via Median ₹/sqft

Instead of one-hot encoding hundreds of locality names, each locality maps to its median price-per-sqft from the dataset. This single numeric captures the entire locational premium — Bandra gets its premium, Nalasopara its suburban rate.

# Smart locality encoding locality_price_map = ( df.groupby('locality') ['price_per_sqft'] .median() .to_dict() ) # 7 features total: # area, beds, baths, balcony, # age, floors, ₹/sqft
03

Feature Scaling

All 7 numeric features are normalised using StandardScaler. This ensures no single feature dominates due to scale differences. The exact scaler fitted on training data is reused at inference time identically.

scaler = StandardScaler() X_scaled = scaler.fit_transform(X) # Persisted to disk pickle.dump(scaler, open("mumbai_scaler.pkl", "wb")) # At inference time: inp_sc = scaler.transform(inp)
04

Random Forest Training

A Random Forest Regressor with 200 estimators and max depth 15 is trained on 80% of data. Averaging predictions from many trees yields extremely high accuracy (R²: 0.9995) while remaining robust against overfitting.

model = RandomForestRegressor( n_estimators=200, max_depth=15, random_state=42 ) model.fit(X_train, y_train) # Test R²: 0.9995
05

Flask API Inference

Model and scaler load once at Flask startup. Form inputs are sent as JSON to /predict, preprocessed identically to training, passed through the model, and returned as a formatted Rupee value — all in milliseconds.

@app.route('/predict',methods=['POST']) def predict(): data = request.get_json() ppsf = loc_map.get(data['locality'], median_ppsf) X = np.array([[area,beds,baths, bal,age,floors,ppsf]]) price = model.predict( scaler.transform(X))[0] return jsonify(format_inr(price))

About the Project

Built for
Mumbai.

EstateIQ is an end-to-end machine learning web application that predicts residential property prices across Mumbai using real listing data. The project demonstrates a complete MLOps pipeline — from raw data ingestion and cleaning, through model training and evaluation, to a production-ready Flask API with a modern frontend.

The goal is simple: give anyone — buyer, seller, or investor — a data-backed starting point for understanding property values across Mumbai's hyperlocal market.
